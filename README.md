# Express-Demo Application

Implemented CRUD REST APIs for courses catalog along with error handling using Express library on Node.
Leveraged Joi library for schema and data validation.

## APIs
- GET /
- GET /api/courses
- GET /api/courses/:id 
- POST /api/courses
- PUT /api/courses/:id
- DELETE /api/courses/:id
